Transaction Behavior Reference
START TRANSACTION

Begins an atomic unit. All subsequent operations are part of this transaction and won't be visible to others until COMMIT.

SAVEPOINT

Creates a named checkpoint within a transaction. Allows partial rollback to the savepoint instead of full transaction rollback.

ROLLBACK

Reverts all changes since the last SAVEPOINT or since START TRANSACTION. Ensures data integrity on errors.
